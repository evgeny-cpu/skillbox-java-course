Технические детали
Примитивы vs Объекты:
Примитивы: хранятся в стеке, имеют фиксированный размер, не имеют методов

Объекты: хранятся в куче, имеют переменный размер, имеют методы

Boxing/Unboxing:
Boxing: автоматическое преобразование примитива в объект (int → Integer)

Unboxing: автоматическое преобразование объекта в примитив (Integer → int)

Тип char:
16-битный беззнаковый целочисленный тип

Хранит символы Unicode

Можно использовать арифметические операции для работы с кодами символов

Примечания
Коды символов зависят от используемой кодировки (в данном случае Unicode)

Буква Ё в Unicode расположена отдельно от основного алфавита

Для корректного отображения русских символов необходима поддержка UTF-8 в консоли

При работе с символами важно учитывать разницу между char и String
