Практическая работа 8.11: Числа и даты
Описание проекта
Данный репозиторий содержит решения двух практических заданий по работе с числами, циклами и датами в Java. 
Работа включает использование автотестов для проверки корректности выполнения.

Структура проекта
Задание 1: Распределение ящиков по грузовикам и контейнерам
Файл: src/main/java/practice/TrucksAndContainers.java

Логика распределения:
Грузовик: вмещает максимум 12 контейнеров

Контейнер: вмещает максимум 27 ящиков

Все элементы пронумерованы

Формат вывода:
text
Грузовик: 1
Контейнер: 1
Ящик: 1
Ящик: 2
Необходимо:
грузовиков - 1 шт.
контейнеров - 1 шт.
Особенности реализации:
Использование целочисленного деления для расчетов

Правильные отступы с помощью символа табуляции \t

Кроссплатформенный перенос строк через System.lineSeparator()

Задание 2: Вывод дней рождения
Файл: src/main/java/practice/Birthdays.java

Метод:
java
public static String collectBirthdays(int year, int month, int day)
Формат вывода (пример):
text
0 - 31.12.1990 - Mon
1 - 31.12.1991 - Tue
2 - 31.12.1992 - Thu
...
Особенности реализации:
Использование класса LocalDate для работы с датами

Сравнение дат методами isBefore() и isAfter()

Правильное форматирование дат и дней недели

Учет високосных годов

Как запустить проект
Предварительные требования:
Java 8 или выше
