Практическая работа 6.8: Статические методы и перечисления
Описание проекта
Данный репозиторий содержит решения двух практических заданий по работе со статическими переменными, методами и перечислениями в Java.

Структура проекта
Задание 1: Модификация класса Basket
Директория: ObjectsAndClasses_Methods/

Изменения в классе Basket:
Добавлены статические переменные:

totalCost - общая стоимость всех товаров во всех корзинах

totalCount - общее количество всех товаров во всех корзинах

Реализованы статические методы:

increaseTotal() - увеличивает значения переменных при добавлении товаров

getAveragePrice() - расчёт средней цены товара во всех корзинах

getAverageBasketCost() - расчёт средней стоимости корзины

Задание 2: Класс Computer и компоненты
Директория: ObjectsAndClasses_StaticFinal/

Классы проекта:
Основной класс:

Computer - класс компьютера с возможностью изменения комплектующих

Иммутабельные классы-компоненты:

Processor - процессор (частота, ядра, производитель, вес)

RAM - оперативная память (тип, объём, вес)

Storage - накопитель (тип, объём, вес)

Screen - экран (диагональ, тип, вес)

Keyboard - клавиатура (тип, подсветка, вес)

Перечисления:

StorageType - HDD, SSD

ScreenType - IPS, TN, VA

Функциональность класса Computer:
Конструктор с параметрами vendor и name

Метод getTotalWeight() - расчёт общей массы компьютера

Геттеры и сеттеры для всех компонентов

Метод toString() - информация о компьютере в удобочитаемом формате

Требования к выполнению
Задание 1:
Созданы обе статические переменные

Реализованы все статические методы

Имена переменных соответствуют хранимым данным

Имена методов соответствуют выполняемым действиям

Код компилируется без ошибок

Задание 2:
Созданы все требуемые классы

Классы-компоненты иммутабельны (поля final)

Реализованы геттеры и сеттеры

Реализован метод расчёта общей массы

Реализован метод toString()

В методе main создано несколько компьютеров

Выводится корректная информация о компьютерах

Как использовать
Для Задания 1:

Перейдите в директорию ObjectsAndClasses_Methods/

Скомпилируйте и запустите проект для тестирования функциональности корзины

Для Задания 2:

Перейдите в директорию ObjectsAndClasses_StaticFinal/

Скомпилируйте и запустите класс Main для просмотра информации о созданных компьютерах

Примечания
Все классы-компоненты в Задании 2 являются иммутабельными

Класс Computer позволяет менять комплектующие через сеттеры

Используются перечисления для типов накопителей и экранов

Реализована проверка на корректность данных в сеттерах
