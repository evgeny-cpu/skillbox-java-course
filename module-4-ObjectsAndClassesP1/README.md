Практическая работа: Классы и методы в Java
Описание проекта
Этот проект содержит решения двух практических заданий по основам объектно-ориентированного программирования в Java. 
В ходе работы реализованы классы с конструкторами, методами и переменными экземпляра.

Структура проекта
Задание 1: Класс Arithmetic
Класс Arithmetic выполняет математические операции с двумя целыми числами.

Функциональность:

Конструктор с двумя параметрами типа int

Методы для вычисления:

Суммы двух чисел

Произведения двух чисел

Максимального из двух чисел

Минимального из двух чисел

Задание 2: Класс Basket
Класс Basket представляет корзину для покупок с возможностью отслеживания общего веса товаров.

Новая функциональность:

Добавлена переменная totalWeight для хранения общей массы товаров

Расширен метод add() для поддержки веса товара

Добавлен метод getTotalWeight() для получения общей массы

Требования
Java 8 или выше

Git для управления версиями

Запуск примеров
Создайте класс с методом main для тестирования функциональности:

java
public class Main {

Arithmetic arithmetic = new Arithmetic(10, 5);
System.out.println("Сумма: " + arithmetic.sum());
System.out.println("Произведение: " + arithmetic.product());
System.out.println("Максимум: " + arithmetic.max());
System.out.println("Минимум: " + arithmetic.min());

        // Тестирование Basket

        basket.add("Яблоки", 100, 2, 0.5);
        basket.add("Молоко", 80, 1, 1.0);
        System.out.println("Общий вес корзины: " + basket.getTotalWeight() + " кг");
    }
}
