Консольное приложение для разделения ФИО
Описание
Консольное приложение на Java, которое разделяет строку с фамилией, именем и отчеством на отдельные компоненты с использованием только методов indexOf, lastIndexOf и substring.

Функциональность
Принимает строку с ФИО в формате "Фамилия Имя Отчество"

Разделяет строку на три компонента: фамилию, имя и отчество

Алгоритм работы
Получает порядковый номер первого пробела с помощью indexOf

Получает порядковый номер второго пробела с помощью lastIndexOf

Извлекает фамилию с помощью substring от начала строки до первого пробела

Извлекает имя с помощью substring от первого пробела до второго пробела

Извлекает отчество с помощью substring от второго пробела до конца строки

Тестовые примеры
Программа протестирована на следующих примерах:

Беллинсгаузен Васисуалий Никифорович

Андреев Дмитрий Иванович

Кольцов Леонид Ильич

Требования
Java Development Kit (JDK) 8 или выше

Ограничения
Для разделения строки используются только методы indexOf, lastIndexOf и substring

Предполагается, что строка содержит ровно три компонента, разделенных пробелами

Дополнительные пробелы в начале или конце строки не обрабатываются
