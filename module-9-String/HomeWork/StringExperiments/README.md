StringExperiments
Проект для экспериментов со строками в Java, демонстрирующий различные способы сравнения строк.

Описание
Программа позволяет пользователю вводить пары строк и сравнивает их различными способами:

Сравнение с учетом регистра (обычное сравнение)

Сравнение без учета регистра

Определение, равны ли строки или различны

Функциональность
Интерактивный ввод: Программа дважды запрашивает у пользователя строки для сравнения

Сравнение строк:

Сравнение с учетом регистра через метод equals()

Сравнение без учета регистра через метод equalsIgnoreCase()

Наглядный вывод: Результаты сравнения выводятся в понятном формате с символами == (равно) и != (не равно)

Технические детали
Основные методы класса String, используемые в проекте:
equals(String anotherString) - сравнивает строки с учетом регистра

equalsIgnoreCase(String anotherString) - сравнивает строки без учета регистра

Полезные советы
Для сравнения строк всегда используйте методы equals() или equalsIgnoreCase(), а не оператор ==

Оператор == сравнивает ссылки на объекты, а не их содержимое

Метод equalsIgnoreCase() полезен при сравнении пользовательского ввода, где регистр может быть произвольным
