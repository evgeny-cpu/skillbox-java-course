Практическая работа по условным операторам
Этот репозиторий содержит решения двух практических заданий по работе с условными операторами и булевой логикой на Java.

Структура проекта
В репозитории находятся два отдельных проекта:

1. Проект Booleans
   Цель: Научиться писать условные операторы и условия из нескольких выражений, объединённых булевыми операторами.

Задача: Написать код, который проверяет, достаточно ли ингредиентов для приготовления различных блюд.

Основной класс: Main.java

Логика работы:

В программе заданы шесть переменных с количествами разных ингредиентов

Реализованы проверки достаточности ингредиентов для различных рецептов

Если ингредиентов достаточно, в консоль выводится название блюда

Используются операторы сравнения >= и логические операторы &&

2. Проект AgeComparator
   Цель: Научиться писать условия из нескольких выражений и объединять их булевыми операторами.

Задача: Написать программу, которая выводит заранее заданные возрасты трёх людей в порядке увеличения.

Основной класс: Main.java

Особенности реализации:

Сравниваются возрасты трёх людей

Определяются минимальный, средний и максимальный возрасты

Код корректно обрабатывает все возможные случаи:

Все возраста различны

Два возраста равны

Все три возраста равны

Возраста могут быть в диапазоне от 0 до 120

Как использовать
Для проекта Booleans:
Откройте проект в среде разработки

Запустите класс Main

Измените значения переменных с ингредиентами для тестирования разных сценариев

Для проекта AgeComparator:
Откройте проект в среде разработки

Запустите класс Main

Измените значения возрастов для тестирования различных комбинаций

Требования
Java Development Kit (JDK) 8 или выше

Среда разработки (IntelliJ IDEA, Eclipse или другая)

Критерии оценки
Задание 1 (Booleans):
✅ Принято: при достаточном количестве ингредиентов выводятся названия соответствующих блюд

❌ На доработку: задание не выполнено или выполнено неточно

Задание 2 (AgeComparator):
✅ Принято: независимо от значений возрастов, они выводятся в порядке увеличения

❌ На доработку: задание не выполнено

Архитектура решения
В проекте Booleans:
Используется подход с последовательными проверками условий

Каждое блюдо проверяется отдельным условным оператором

Логические выражения объединяют проверки нескольких ингредиентов

В проекте AgeComparator:
Реализовано сравнение всех возможных комбинаций возрастов

Используются вложенные условные операторы

Учтены все возможные случаи равенства возрастов
